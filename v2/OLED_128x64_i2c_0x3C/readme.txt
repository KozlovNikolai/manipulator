миниатюрный дисплей около 24 мм диагональ 128х64 точки интерфес i2c 

1) в Arduino IDE открыть - скеч - Подкл библ - Добавить ZIP-библ
 и добавить  Adafruit_GFX.zip  а затем Adafruit_SSD1306.zip
 
2) подключите дисплей к Arduino через преобразователь уровей на 3.3 вольта (например через модуль согласования логических уровней)
 
3) в Arduino IDE открыть - Файл - открыть -  откройте скеч
C:\Users\......\Documents\Arduino\libraries\Adafruit_SSD1306\examples\ssd1306_128x64_i2c/ssd1306_128x64_i2c.ino

4) нажать кнопку "загрузить" - после компиляции и загрузки на дисплее должна появится демка
